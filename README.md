# ikonfilm

A minimal photography portfolio for Ira Kaur.

## Upload to GitHub

1. Create a repository named `ikonfilm`.
2. Upload these files into the repository root:
   - `index.html`
   - `style.css`
   - `script.js`
   - `images/`
3. Put your photos in the `images` folder.
4. Turn on GitHub Pages from `main` and `/ (root)`.

## Notes

- The collections section saves to the browser on that device using localStorage.
- Replace `hello@yourdomain.com` and `@yourhandle` in `index.html`.
- The featured images currently point to files in the `images` folder.
